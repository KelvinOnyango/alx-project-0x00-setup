# Airbnb Clone - Components Setup

## Current Components

1. **Card Component**

   - Displays property information with:
     - Image
     - Title and location
     - Rating
     - Amenities
     - Price

2. **Pill Component**
   - Reusable badge component for tags
   - Takes a `title` prop to display different labels

## Running the Project

```bash
npm run dev -- -p 3000
```
